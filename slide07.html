TA-CU

기술적 도전과 해결

1

카카오맵 SDK 로드 타이밍

`window.kakao` 객체가 정의되기 전에 지도 초기화를 시도하여 오류 발생. React의 렌더링 속도와 SDK 로드 속도 불일치.

재시도 로직 구현

250ms 간격으로 SDK 로드 여부를 확인하는 재시도 로직(최대 10초)을 추가하여 안정적인 지도 초기화 보장.

2

타슈 API 인증 방식 불일치

기존 코드는 쿼리스트링 방식을 사용했으나, 실제 API는 헤더(`api-token`) 인증만 허용하여 401 Unauthorized 오류 발생.

헤더 인증 및 파서 재작성

인증 방식을 Header로 변경하고, 응답 데이터 구조(`results[]`)에 맞춰 파서를 전면 수정하여 데이터 연동 성공.

3

GPS 신호 튀는 현상

도심 빌딩 숲에서 GPS 오차가 발생하여 실제 이동하지 않은 거리까지 계산되거나 경로가 지그재그로 표시됨.

Accuracy 필터링 적용

브라우저가 제공하는 `accuracy`(정확도 반경) 값이 30m 이상인 저정밀 신호는 기록에서 자동 제외하여 데이터 품질 향상.
